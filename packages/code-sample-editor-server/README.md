# lit-code-editor-server
Server that runs the preview of the `lit-code-editor` element.

## Getting Started
### development
```sh
# in lit-code-editor-package root directory
npm install
npm run bootstrap
npm run build
cd packages/lit-code-editor-server
npm run dev
```

### usage
```sh
# in lit-code-editor-package root directory
npm install
npm run bootstrap
npm run build
cd packages/lit-code-editor-server
npm start
```
